# Kuikly Roadmap (2026)

`Kuikly` 是腾讯内外广泛使用的跨端框架，基于 Kotlin Multiplatform 技术，为开发者带来技术栈更一致的全新跨端开发体验。2025 年我们完成了鸿蒙、H5、小程序平台的开源，发布了 Compose DSL Beta 版本，并在社区建设上迈出了重要一步。

**2026 年，Kuikly 将围绕「AI 驱动开发」这一核心方向展开**——让 AI 更好地理解 Kuikly，让开发者借助 AI 更高效地构建跨端应用。同时，我们将继续推进 Compose DSL 与多平台能力的成熟度，并加大生态建设的投入。

## 一、AI 驱动开发（核心方向）

AI 正在深刻改变软件开发方式。2026 年，我们的目标是让 Kuikly 成为**对 AI 最友好的跨端框架之一**，将 AI 能力深度融入开发全链路。围绕这一目标，我们将重点推进以下方向：

- **AI 基础设施**：通过 MCP Server、Rules、Skills、AI 知识库等能力建设，让主流 AI 编程工具深度理解 Kuikly 框架，实现高质量的智能代码生成与补全。
- **智能问答助手**：打造 Kuikly 专属的 AI 问答能力，为开发者提供框架使用咨询、问题排查与最佳实践推荐，降低上手门槛与日常答疑成本。
- **设计与转码**：支持设计稿（Figma）到 Kuikly 代码的自动转化，同时提供存量代码（React / Vue / Flutter 等）到 Kuikly 的智能转码能力，降低迁移成本。
- **预览与调试**：打通 AI 生成代码到即时预览的闭环体验，配合 UI Inspector 实现可视化调试，大幅缩短"编码 → 验证"周期。
- **AI 应用组件**：提供开箱即用的 AI Chat 等组件，帮助开发者快速构建 AI 驱动的跨端应用。

## 二、Compose DSL 与平台支持

### 2.1 Compose DSL 正式发布

Compose DSL 在 2025 年已经过腾讯内部多个业务的大规模验证。2026 年将推进其正式 Release。

| 里程碑 | 说明 |
|-------|------|
| **Release 发布** | 完成 Compose DSL 的稳定性收敛与 API 冻结，发布正式版本 |
| **API 对齐度提升** | 持续提升与官方 Compose API 的对齐度，进一步降低迁移和学习成本 |
| **工具链完善** | 完善 Compose 预览、性能工具（Recompose 追踪）等开发辅助工具 |

### 2.2 多平台 Release 推进

将目前处于 Beta / Alpha 状态的平台推向正式发布。

| 平台 | 当前状态 | 2026 目标 |
|------|---------|----------|
| **Web（H5）** | Beta | 正式 Release |
| **微信小程序** | Beta | 正式 Release |
| **macOS** | Alpha | Beta Release |

## 三、性能与工程优化

性能是跨端框架的核心竞争力。2026 年，我们将在编译速度、运行性能、内存占用、包体积等方面进行系统性优化，持续提升开发与运行体验：

| 方向 | 说明 |
|------|------|
| **编译速度** | 通过 klib 增量编译、远程编译、kotlinc 增量编译等手段，大幅缩短编译耗时 |
| **运行性能** | 引入 CSR、Wasm 优化、TurboDisplay 等技术，显著提升启动速度与页面流畅度 |
| **内存优化** | 利用系统特性加速特定场景释放内存效率、探索落地CMC 内存整理等方案，有效降低内存占用 |
| **包体积** | 推进 KMM 减包与拆包能力，结合动态加载技术，减小产物体积 |
| **稳定性保障** | 建设白屏监控等异常监控机制，提升线上问题发现与定位效率 |

## 四、生态建设

一个繁荣的生态是框架长期发展的基石。2026 年我们将在以下方面加大投入：

| 方向 | 说明 |
|------|------|
| **社区组件平台** | 持续丰富社区组件库，完善组件发布、发现与使用流程，引入质量评级与安全审查机制，鼓励社区贡献与共建 |
| **三方库兼容与适配** | 推进与 KMP 生态中主流三方库的兼容适配，降低集成成本 |
| **文档与教程** | 根据社区反馈持续优化文档质量，提供更多实战教程和最佳实践 |
| **开发者社区** | 维护活跃健康的社区交流氛围，完善贡献者激励机制，举办技术分享和线上活动 |

## 总结

2026 年，Kuikly 将以 **AI 驱动开发** 为核心方向，从 AI 基础设施、设计与转码、预览调试到 AI 应用组件，全链路释放跨端开发生产力；同时推进 Compose DSL 正式发布、Web/小程序/macOS 全面 Release；在性能与工程优化方面，系统性提升编译速度、运行性能、内存与包体积表现；并持续投入生态建设，与社区开发者共同打造更好的跨端开发体验。

期待与社区的优秀开发者一起携手，共创 Kuikly 的 AI 时代！

# 社区分享

👏👏👏 欢迎来到``Kuikly``社区分享板块，本板块的核心目的是促进技术分享和共同成长。

开发者的个人项目实践经验是社区最宝贵的财富。我们鼓励每一位开发者参与进来，无论是发表文章、回复问题，还是参与讨论。通过分享自己的见解，并借鉴他人的经验，我们可以互相学习，共同进步。

## 社区分享汇总

**社区分享平台网址**：https://kuikly.tds.qq.com/third-party?tab=artical

## 社区分享收集

知识分享采用去中心化分享，中心化收集展示的管理方式，这样既方便贡献者通过自己的渠道分享，也方便社区开发者查阅。

1. 首先在自己的渠道，比如个人博客、三方技术社区(掘金、CSDN、知乎等)发表文章。
2. 然后将三方社区的分享，按照[Kuikly-third-party](https://github.com/Tencent-TDS/KuiklyUI-third-party)指引，提交到``Kuikly``社区。
